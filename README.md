# Workflow Management Application

To run the backend locally, navigate to the backend directory, install the dependencies, and start the server.

cd backend  
npm install  
npm start  

The backend server runs on http://localhost:5000

To run the frontend locally, navigate to the frontend directory, install the dependencies, and start the development server.

cd frontend  
npm install  
npm run dev  

The frontend server runs on http://localhost:5173
